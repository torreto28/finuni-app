# FinUni: Academic Wealth

## Descripción
FinUni es una aplicación financiera integral diseñada específicamente para el ecosistema universitario. Este proyecto, desarrollado como parte de la formación en ingeniería en la **Universidad de Medellín**, busca fusionar el control de gastos diario con un radar de inversión contextual (ETFs, CEDEARs, Fondos de liquidez).

El objetivo es permitir a los estudiantes construir hábitos financieros robustos desde una etapa temprana, eliminando las barreras de los brókers tradicionales mediante una interfaz intuitiva y datos en tiempo real.

## Metodología y Estrategia
El diseño de FinUni se fundamenta en un análisis técnico riguroso utilizando metodologías de innovación:

* **PNI (Positivo, Negativo, Interesante)**: Evaluamos el impacto de la herramienta, desde la accesibilidad tecnológica hasta la educación financiera predictiva[cite: 1, 2, 8].
* **SCAMPER**: Aplicado para transformar la gestión financiera tradicional.
    * **Sustituir**: Gráficos complejos por "Análisis de Bolsillo"[cite: 22, 23].
    * **Combinar**: Calendario académico con recomendaciones de inversión pasiva[cite: 29].
    * **Adaptar**: Uso de "Cajones digitales" para mi primer activo[cite: 32].
    * **Eliminar**: La fricción de mover dinero a plataformas externas[cite: 44].
    * **Reordenar**: Prioridad a "Invertir Primero, Gastar Después"[cite: 49].

## Características Técnicas
* **Arquitectura Modular**: Backend desarrollado con Node.js y Express, con frontend independiente para facilitar la escalabilidad.
* **Estética**: Diseño bajo estándares de "Academic Wealth" (Paleta: Verde Bosque, Oro Champagne, Pergamino).
* **Funcionalidad**: Incluye "Pocket Analysis" para detección de gastos hormiga e inversión automática basada en excedentes.
